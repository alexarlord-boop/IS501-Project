## Prototypes

Open `code.html` from each subfolder directly in a browser — no server required. Each prototype includes a `screen.png` screenshot alongside the live HTML.

| Folder | Actor | Covers |
|--------|-------|--------|
| `stitch prototype/customer_order_page/` | Customer | Browse menu, configure cake, checkout, loyalty points |
| `stitch prototype/baker_dashboard/` | Baker | Order queue, status updates, stock alerts |
| `stitch prototype/mobile_delivery_app/` | Delivery Staff | Daily route, stop details, mark delivered |
